## 一、项目概述

### 1.1 项目背景

时光树是一款贯穿用户全生命周期的AI成长伙伴产品，通过「公私双空间隔离架构 + 多成长线人生时间轴 + 专属终身知识库 + 长期记忆AI + 年龄适配UI系统」，为用户提供从孕期到终老的全程陪伴。

### 1.2 技术架构

| 端 | 技术选型 | 职责 |
|----|----------|------|
| 移动端 | UniApp + Vue 3 | 用户主入口，跨平台iOS/Android/H5/小程序 |
| 后端Go | Go 1.22+ + Gin | 核心业务API、用户体系、家庭管理、时间轴 |
| 后端Python | Python 3.12+ + FastAPI | AI能力服务、记忆引擎、情感分析 |
| 管理后台 | Next.js 14 + TypeScript | 运营管理、插件管理、用户管理 |
