## 十一、API 接口规划

### 11.1 用户与认证

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/v1/auth/sendCode | POST | 发送验证码 |
| /api/v1/auth/login | POST | 验证码登录 |
| /api/v1/auth/wechat | POST | 微信登录 |
| /api/v1/user/profile | GET | 获取用户信息 |
| /api/v1/user/profile | PUT | 更新用户信息 |

### 11.2 生命管理

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/v1/life | POST | 创建生命 |
| /api/v1/life | GET | 获取当前生命 |
| /api/v1/life/list | GET | 获取用户所有生命 |
| /api/v1/life/{id}/stage | PUT | 更新生命阶段 |

### 11.3 家庭管理

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/v1/family | POST | 创建家庭 |
| /api/v1/family | GET | 获取家庭信息 |
| /api/v1/family/members | GET | 获取家庭成员 |
| /api/v1/family/invite | POST | 生成邀请码 |
| /api/v1/family/join | POST | 加入家庭 |

### 11.4 时间轴

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/v1/timeline | GET | 获取时间轴列表 |
| /api/v1/timeline | POST | 发布时间轴 |
| /api/v1/timeline/{id} | GET | 时间轴详情 |
| /api/v1/timeline/{id} | DELETE | 删除时间轴 |

### 11.5 内容管理

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/v1/content/upload | POST | 上传内容 |
| /api/v1/content/{id} | GET | 获取内容 |
| /api/v1/content/{id} | DELETE | 删除内容 |

### 11.6 AI 服务

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/v1/ai/chat | POST | AI对话 |
| /api/v1/ai/summary | POST | 内容摘要 |
